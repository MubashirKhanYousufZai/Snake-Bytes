## What is Python?

Python is a **high-level**, **interpreted**, and **general-purpose programming language** created by **Guido van Rossum** and released in **1991**.

It was designed with one main philosophy:

> Code should be **easy to read**, **easy to write**, and **easy to maintain**.

Python emphasizes simplicity and clarity, which makes it ideal for:
- Beginners learning programming
- Professionals building production systems
- Rapid prototyping and experimentation
