## What is Python?

Python is a **high-level**, **interpreted**, and **general-purpose programming language** created by **Guido van Rossum** and released in **1991**.

It was designed with one core philosophy:

> Code should be **easy to read**, **easy to write**, and **easy to maintain**.

Unlike many older programming languages, Python focuses on:
- Human-readable syntax
- Minimal boilerplate code
- Logical structure over symbols

Because of this, Python has become a favorite among beginners and professionals alike.
